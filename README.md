# WEATHER APP

## About the app

I have built the weather app with Javascript, hbs, Node.js and Express following Udemy's course 'The Complete Node.js DEveloper Course (3rd Edition)' instructed by Andrew Mead.

## Features

- weather API (mapbox)
- Browser HTTP request
- Search form
